## Response


我来帮你画几个常用的Mermaid图示例，你可以选择需要的类型：

## 1. **流程图 (Flowchart)**
```mermaid
graph TD
    A[开始] --> B{条件判断}
    B -->|是| C[执行操作]
    B -->|否| D[其他操作]
    C --> E[结束]
    D --> E
```

## 2. **时序图 (Sequence Diagram)**
```mermaid
sequenceDiagram
    participant 用户
    participant 系统
    participant 数据库
    
    用户->>系统: 发送请求
    系统->>数据库: 查询数据
    数据库-->>系统: 返回结果
    系统-->>用户: 显示响应
```

## 3. **类图 (Class Diagram)**
```mermaid
classDiagram
    class 动物 {
        +String 名称
        +void 移动()
    }
    
    class 鸟 {
        +void 飞行()
    }
    
    class 鱼 {
        +void 游泳()
    }
    
    动物 <|-- 鸟
    动物 <|-- 鱼
```

## 4. **甘特图 (Gantt Chart)**
```mermaid
gantt
    title 项目时间表
    dateFormat YYYY-MM-DD
    
    section 设计
    需求分析 :2024-01-01, 7d
    系统设计 :2024-01-08, 5d
    
    section 开发
    前端开发 :2024-01-15, 10d
    后端开发 :2024-01-15, 12d
    
    section 测试
    单元测试 :2024-01-25, 5d
    集成测试 :2024-01-30, 5d
```

## 5. **饼图 (Pie Chart)**
```mermaid
pie title 市场份额
    "公司A" : 35
    "公司B" : 25
    "公司C" : 20
    "其他" : 20
```

## 6. **状态图 (State Diagram)**
```mermaid
stateDiagram-v2
    [*] --> 待机
    待机 --> 运行: 启动
    运行 --> 暂停: 暂停按钮
    暂停 --> 运行: 继续按钮
    运行 --> 待机: 停止
    暂停 --> 待机: 停止
```
