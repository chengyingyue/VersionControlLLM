# VC-LLM 用户使用指南

欢迎使用 VC-LLM，这是一个基于 Markdown 存储的、具备版本控制能力的对话系统。

## 1. 快速开始

1. **启动服务**：确保后端服务已运行 (`python -m app.main`)。
2. **访问界面**：在浏览器打开 `http://localhost:8002`。
3. **用户登录**：
   - 在登录页面输入你的 **工号** 或 **姓名**。
   - 点击“登录”进入主界面。
4. **新建对话**：点击侧边栏的“新建对话”按钮，输入对话名称和初始 System Prompt。

## 2. 核心功能操作

### 2.1 聊天与流式生成
- 在底部输入框输入内容并回车。
- 机器人回复将实时流式显示。
- **停止生成**：如果回复太长或不符合预期，点击输入框上方的“停止”按钮。

### 2.2 历史重写 (Rewrite)
- 悬停在任何一条历史消息（User 或 Assistant）上。
- 点击出现的“编辑”图标。
- 修改内容并提交。
- **注意**：重写操作会删除该消息之后的所有对话，并基于新内容重新生成。

### 2.3 对话分叉 (Fork)
- 在顶部工具栏点击“Fork”按钮。
- 系统会复制当前对话的所有历史到一个新对话中。
- 你可以在新分支上尝试不同的提问思路，而不会影响原对话。

### 2.4 修改 System Prompt
- 在顶部工具栏点击“System Prompt”。
- 修改 Message 0 的内容。
- 这将影响后续的所有生成逻辑，但不会删除已有历史。

### 2.5 回滚 (Rollback)
- 点击顶部工具栏的“回滚”或通过历史记录操作。
- 选择要回退到的历史点，系统将截断后续内容。

## 3. 高级特性

### 3.1 Mermaid 支持
- 如果 AI 返回了 `mermaid` 代码块，系统会自动将其渲染为流程图、序列图等可视化图表。

### 3.2 源码导出
- 点击“Export”可以获取当前对话的完整 Markdown 源码，方便复制到其他地方或进行备份。
